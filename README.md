# Morse Code Rush!
a fast paching morse code practicing site when you can learn and practice morse code!

## how to use it
Just short press space to write (.) 
and long press space to write dash (-)
there is also a handy lookup table for morse code at the bottom!

## features
* minimal UI
* handy lookup table!
* thats literary it its to practice your morse code writing skill! (thinking about it right now i might add a morse code reading part in future)
